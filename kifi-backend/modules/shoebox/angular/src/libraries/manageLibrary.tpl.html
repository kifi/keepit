Slack

It's easy to push updates from this library to one of your company's Slack channels. To configure the integration between Kifi and Slack:

  1. Log into Slack, then go to this page.
  2. Choose a default channel, then click "Add Incoming WebHooks Integration"
  3. Copy the Webhook URL, and paste it in the URL fields below.
  4. Enter the name of the #channel or @mention you want updates sent to, then "Save".

All new keeps added to this library will now appear in Slack!
